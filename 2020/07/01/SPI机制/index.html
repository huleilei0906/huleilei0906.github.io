1.1 SPI思想

SPI的全名为Service Provider Interface.这个是针对厂商或者插件的。

1.2 SPI约定

1.2.1 Java

当服务的提供者，提供了服务接口的一种实现之后，在jar包的META-INF/services/目录里创建一个以服务接口命名的文件。

1.2.2 Spring
